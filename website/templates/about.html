About The Project

Current Goals

This site is a front end for a butterfly classification system developed under the supervision of Doctor Benjamin Glocker at Imperial College London as part of my MSc in Computing Science. Using a web based front end allows for flexibility and easy cross platform support. The original project proposal is included below for completeness but in its current form, the project is very different. Namely I have decided to build the identification system which uses a bag of words with spatial pyramids approach to be as flexible as possible, focusing on the framework over the specific system and original application. The end product will eventually be made public to allow others to build simple robust identification systems with minimal effort.


Description of project as proposed by B. Glocker

The goal of this project is to develop a software tool for automatic identification of fossils and insects from photographs. Such a tool could greatly support taxonomists in classification of existing and discovery of new species. The automatic identification will be achieved by implementing a robust image processing pipeline with takes elements from pattern recognition and machine learning into account. Based on discriminative feature extraction and supervised clustering techniques the software should be able to classify a given photograph with high accuracy.

The software will be implemented either in Matlab or C++ using the OpenCV library (depending on the student's expierence). Knowledge in image processing, computer vision, and machine learning/artificial intelligence are beneficial.

This project is in collaboration with Prof. Norman MacLeod from the National History Museum London.
